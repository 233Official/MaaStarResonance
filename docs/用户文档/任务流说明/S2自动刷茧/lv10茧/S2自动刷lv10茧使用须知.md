# S2自动刷Lv10茧使用须知

写了个潦草的版本挂Lv10的浮游之茧,
可以传送到最近的这个传送点这里:

![image-20251104001638920](S2自动刷lv10茧使用须知.assets/image-20251104001638920.png)

随便找一页快捷栏把这个传送点放在正下方这个位置:

![image-20251103220211406](S2自动刷lv10茧使用须知.assets/image-20251103220211406.png)

记得手动把这个一键连招勾上, 不然UI上不显示

![c559c6e0c252e1776806774c470dbb0b](S2自动刷lv10茧使用须知.assets/c559c6e0c252e1776806774c470dbb0b-1762887248653-3.png)
