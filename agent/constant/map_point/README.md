
# 地图传送点坐标配置文件

## 模拟器分辨率 `1280 x 720` 平板模式

## 字段说明

| 字段名 | 含义 |
| :--: | :--: |
| x | 左上角为0开始的横坐标 |
| y | 左上角为0开始的纵坐标 |
| floor | 地图楼层按钮对应的坐标 |
| alias | 传送点按钮名称别称 |

### 注意

1. 因为比如高原是分上下两层的，在切换地图的时候需要确定是哪个楼层，所以才需要传入`floor`做为右侧切换楼层按钮的坐标
2. `alias`一般不用，只有传送点名字和按钮名不一样的时候才用，比如`传送-崖之遗迹`的按钮叫`星元枢纽`，还有部分同名传送点也可以这样区别
3. 目前传送点还未添加完，需要可以自行添加至`MapPoint.json`

## 样例

```json
{
    "阿斯特里亚平原": {
        "传送-安德拉废都": {
            "x": 511,
            "y": 219
        },
        "传送-崖之遗迹": {
            "x": 613,
            "y": 562,
            "alias": "星元枢纽"
        }
    },
    "巴哈马尔高原": {
        "传送-巡礼者的休憩之所": {
            "x": 671,
            "y": 468,
            "floor": {
                "x": 1230,
                "y": 330
            }
        },
        "传送-贝克马尔运输站-大点": {
            "x": 767,
            "y": 382,
            "alias": "传送-贝克马尔运输站",
            "floor": {
                "x": 1230,
                "y": 330
            }
        },
        "传送-贝克马尔运输站-小点": {
            "x": 767,
            "y": 382,
            "alias": "传送-贝克马尔运输站",
            "floor": {
                "x": 1230,
                "y": 330
            }
        }
    }
}
```
