## 功能列表

- 自动钓鱼
- 简易采集
- 领取今日活跃度奖励
- 挂机自动批复入队申请
- 每130s释放一次一号位幻想
- 聊天频道发言
- 周期性聊天频道发言
- Lv10浮游之茧
- 启动/关闭/重启游戏
- 启动并登录星痕共鸣
- 自动协会狩猎
- 打开/关闭一键连招
- 从省电模式唤醒
- 关闭所有广告弹窗(beta)
- 地图传送
- 地图导航
